## Clone Repository
```
git clone https://github.com/octra-labs/wallet-gen.git
```
```
cd wallet-gen
```
## Install Bun
```
curl -fsSL https://bun.sh/install | bash
```
```
source ~/.bashrc
```
```
bun install
```
```
bun run build
```
```
bun start
```
- A pop-up will appear use it to open the local host browser
